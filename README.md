## Run 
```bash
$ git clone https://github.com/bogoslavskiy/react-native-score-app.git
$ cd react-native-score-app
$ yarn install

# With expo 
$ npm install --global expo-cli # if not installed

$ yarn ios
$ yarn android

# Without expo 
$ cd ios
$ pod install
$ open scoreapp.xcworkspace

```
